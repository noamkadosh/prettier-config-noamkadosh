# Installation

**npm**

```Shell
npm install --save-dev prettier-config-noamkadosh
```

**yarn**

```Shell
yarn add -D prettier-config-noamkadosh
```

**pnpm**

```Shell
pnpm add -D prettier-config-noamkadosh
```

# Usage

In a `.prettierrc.js` file:

```JavaScript
module.exports = 'prettier-config-noamkadosh'
```
